# 🚀 考研英语生词本 - 快速开始指南

## 📋 项目概览

这是一个专为考研英语二学生设计的生词学习应用，具有以下核心功能：

- 📚 **单词管理** - 录入、分类、搜索生词
- 🤖 **AI智能分析** - 自动分析考研释义、熟词僻义
- 🎯 **三种背诵模式** - 卡片、听音、测验
- 📊 **学习统计** - 进度跟踪、正确率分析
- ⚙️ **个性化设置** - 自定义学习计划

## 🎯 核心功能演示

### 1. 添加单词
1. 点击首页的"添加单词"按钮
2. 输入单词（支持AI智能分析）
3. 选择分类（阅读/完型/翻译/作文）
4. 设置难度等级（1-5星）

### 2. 开始背诵
1. 点击"开始背诵"按钮
2. 选择学习模式：
   - 📖 **卡片模式** - 翻转卡片学习
   - 🔊 **听音模式** - 听发音写单词
   - ✏️ **测验模式** - 选择题练习

### 3. 查看统计
1. 进入"统计"页面
2. 查看学习进度和正确率
3. 识别困难单词
4. 追踪一周学习趋势

### 4. 管理单词
1. 进入"单词本"页面
2. 浏览所有单词
3. 按分类筛选
4. 搜索特定单词
5. 编辑或删除单词

## 📱 使用场景

### 场景1：遇到生词
1. 在真题中遇到不认识的单词
2. 打开App，点击"添加单词"
3. 输入单词，AI自动分析考研释义
4. 选择合适的分类和难度
5. 单词自动加入学习计划

### 场景2：日常背诵
1. 每天固定时间打开App
2. 查看今日学习任务
3. 选择合适的学习模式
4. 完成学习后查看统计
5. 根据正确率调整学习策略

### 场景3：考前冲刺
1. 查看"困难单词"列表
2. 重点复习熟词僻义
3. 使用"测验模式"模拟考试
4. 跟踪学习进度和正确率
5. 调整复习重点

## 💡 学习建议

### 最佳实践
1. **每日坚持** - 建议每天学习20-30分钟
2. **循序渐进** - 从简单单词开始，逐步提高难度
3. **重点突破** - 多复习"困难单词"列表中的词
4. **模式切换** - 不同场景使用不同学习模式
5. **定期回顾** - 利用艾宾浩斯复习计划

### 考研策略
1. **分类学习** - 按阅读、完型、翻译、作文分类
2. **熟词僻义** - 特别关注一词多义的情况
3. **真题导向** - 优先学习真题中出现的高频词
4. **语境记忆** - 结合例句理解单词用法
5. **定期测试** - 使用测验模式检验学习效果
