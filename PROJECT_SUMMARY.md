## 🏗️ 技术架构

### 前端技术栈
- **框架**: React Native + Expo SDK 50
- **UI组件库**: React Native Paper 5.11
- **导航**: React Navigation 6.x
- **状态管理**: React Hooks
- **类型系统**: TypeScript
- **数据存储**: AsyncStorage (轻量级)

### 后端服务
- **AI服务**: OpenRouter API (支持Claude、GPT等多模型)
- **本地存储**: AsyncStorage
- **语音服务**: expo-speech
- **日期处理**: date-fns

### 核心算法
- **艾宾浩斯遗忘曲线**: [1, 2, 4, 7, 15, 30] 天复习间隔
- **智能复习调整**: 基于正确率动态调整复习频率
- **单词难度评估**: 1-5级难度系统
- **学习计划生成**: 个性化每日学习任务

## 🎨 核心功能设计

### 1. 生词管理
- **手动录入**: 支持单个单词输入
- **批量导入**: AI自动提取真题生词
- **智能分类**: 阅读、完型、翻译、作文
- **熟词僻义**: 特别标注考研重点

### 2. AI智能辅助
- **考研专属释义**: 只显示考研常考含义
- **形近词提醒**: 自动关联易混淆单词
- **词根词缀分析**: 帮助记忆和理解
- **真题例句**: 提供考研风格例句

### 3. 学习计划
- **艾宾浩斯算法**: 科学安排复习时间
- **个性化调整**: 根据掌握程度调整难度
- **每日任务**: 新词学习 + 旧词复习
- **进度追踪**: 实时学习数据统计

### 4. 背诵模式
- **卡片翻转**: 单词 ↔ 释义切换
- **听音背词**: 语音辅助记忆
- **错题收集**: 自动收录错误单词
- **多种题型**: 选择、填空、翻译

## 🛠️ 部署和运行

### 开发环境
```bash
# 安装依赖
npm install

# 配置环境变量 (创建 .env 文件)
OPENROUTER_API_KEY=your_api_key

# 启动开发服务器
npx expo start
```

### 生产部署
```bash
# Web部署
npx expo export:web

# 移动端打包
npx expo build:ios
npx expo build:android
```
