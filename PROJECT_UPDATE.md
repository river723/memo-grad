## 🔧 技术栈

### 前端技术
- **框架**: React Native + Expo SDK 50
- **UI库**: React Native Paper 5.11 (Material Design)
- **导航**: React Navigation 6.x
- **类型系统**: TypeScript 5.3
- **存储**: AsyncStorage
- **图标**: react-native-vector-icons
- **语音**: expo-speech
- **日期处理**: date-fns

### 核心功能特性
- **单词管理**: 增删改查、分类管理、搜索筛选
- **AI分析**: 考研释义、熟词僻义标注、形近词提醒
- **学习模式**: 卡片背诵、听音背词、考题练习
- **智能算法**: 艾宾浩斯记忆曲线、个性化学习计划
- **数据统计**: 学习进度、正确率分析、可视化图表
- **个性化设置**: 每日新词数量、发音设置、主题配置

### 启动说明
由于Windows系统上的Metro配置问题，建议使用以下方式测试：

```bash
# 清除缓存并重新安装依赖
rm -rf node_modules package-lock.json
npm install

# 创建Metro配置文件
echo "const { getDefaultConfig } = require('expo/metro-config');\n\nconst config = getDefaultConfig(__dirname);\nmodule.exports = config;" > metro.config.js

# 启动应用
npx expo start --web
```
