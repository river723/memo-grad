## 🎨 技术架构总结

### 前端技术栈
- **框架**: React Native + Expo SDK 50
- **UI库**: React Native Paper 5.11
- **导航**: React Navigation 6.x
- **状态管理**: React Hooks
- **类型系统**: TypeScript 5.3
- **存储**: AsyncStorage
- **网络**: Axios

### 后端服务
- **AI服务**: DeepSeek API
- **语音服务**: expo-speech
- **日期处理**: date-fns

### 核心算法
- **艾宾浩斯记忆曲线**: [1,2,4,7,15,30]天复习间隔
- **智能复习调整**: 基于正确率动态调整
- **个性化学习**: 根据用户表现定制计划
